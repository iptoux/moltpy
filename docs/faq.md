# FAQ

**Where is my data stored?**

In the Moltpy data directory, default: `C:\Users\<Name>\.moltpy`.

**How do I change the storage location?**

Set `override.moltpy_path` in `moltpy.json`.

**How do I exit Moltpy cleanly?**

`exit` or `quit` in the console. Alternatively `stop`.

**How do I reload configuration?**

`reload` in the console.

**How do I see current status?**

`status` for the short view or `status full` for details.
