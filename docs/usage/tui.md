# TUI Usage

The Moltpy console is a text-based interface with three areas.

**Layout**

- Header: name, short description, status, thread info, and uptime
- Center: output and messages
- Footer: input line and hints

**Input**

- Type a command and press Enter.
- Tab completes commands and arguments.
- Up/Down arrows browse command history.
- PageUp/PageDown scroll the output.

**Tips**

- `help` shows all commands.
- `status full` provides a detailed status summary.
- `tools` lists loaded tools and descriptions.
