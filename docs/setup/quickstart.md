# Quickstart

This page walks you through the first run of Moltpy.

**Start**

```powershell
python src\moltpy.py
```

After starting, the interactive console opens.

**On first run**

- A Moltpy data directory is created.
- Default files for configuration, data, and logs are prepared.

**Exit**

- Enter `exit` or `quit`.
- Alternatively: `stop` ends the run.
